# Repository Skills

These are local, reusable workflows for agents operating in this repository.

## Available Skills
- `skills/pickup-preflight/SKILL.md`
  - Validate role-eligible pickup from fresh backlog state.
- `skills/index-sync/SKILL.md`
  - Sync `backlog/index.md` row after issue gate/status changes.
- `skills/openspec-change-bootstrap/SKILL.md`
  - Create a new OpenSpec change package from templates.

Use these skills to reduce repeated reasoning tokens and prevent state drift.
