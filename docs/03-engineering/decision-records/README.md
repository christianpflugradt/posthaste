# Architecture Decision Records

Store ADRs in this directory as `ADR-001-<slug>.md`, `ADR-002-<slug>.md`, etc.

## Rules
- Create ADRs for decisions that impact architecture, interfaces, quality gates, or operational risk.
- One decision per ADR.
- Update status to `Superseded` when replaced; link replacement ADR.
