# Product One-Pager

## Product
`posthaste` is a tool that suggests relevant LinkedIn hashtags from user-provided post text.

## Target User
People who publish on LinkedIn and want better discoverability with less manual effort.

## Core Outcome
Given input text, generate concise, relevant, and quality hashtags suitable for LinkedIn.

## Non-Goals
- Full social media scheduling platform.
- Automated posting on behalf of users (initially).
- Engagement analytics suite (initially).
