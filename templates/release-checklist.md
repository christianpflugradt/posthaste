# Release Checklist

## Scope
- [ ] Scope matches approved RFC.
- [ ] Changelog/release notes prepared.

## Quality
- [ ] Tests pass.
- [ ] Evaluation gates pass.
- [ ] Review completed; no open blocking findings.

## Operations
- [ ] Rollback procedure confirmed.
- [ ] Monitoring/verification steps defined.

## Approval
- [ ] Stakeholder go/no-go recorded.
